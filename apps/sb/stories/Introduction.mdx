# HyezoUI

<div className="subheading">한글소개</div>
안녕하세요. `hyezoUI`는 저의 `design system` 입니다.  
`Next.js`, `TailwindCSS`, `Zod` 를 기반하여 채워가고 있습니다.  
개인적으로 활용하기 위해 확장성과 편의성을 동시에 고민하며 만들고 있어요.

<div className="subheading">패키지</div>
모든 컴퍼넌트는 라이브러리 [`hyezo`](https://www.npmjs.com/package/hyezo?activeTab=explore)에 담아 배포하고 있습니다.  
빌드에 사용된 커스텀 훅 또한 포함되어 있으며,  
번들링에는 `tsup`을 활용하였습니다.

```ts install
pnpm i hyezo

import "hyezo/ui/styles.css"   // 설치 후 root파일에서(_app.tsx)
```
